## Installation

### Clone this repository

```
git clone "https://github.com/victorzix/stock"
cd [FOLDER NAME]
cd backend
```

### Install dependencies

```
npm install
```

### Run the application

```
npm run dev
```

# Endpoints
- # Store
  `POST /products`
  ## Request example
  ```json
  {
    "name": "Cookie",
    "sector": "1",
    "price": 5,
    "quantity": 300
  }
  ```
  
  ## Response example
  `201 Created`
  ```json
  "message": "Product successfully created",
	"data": {
		"name": "aasasd",
		"sector": "2",
		"price": 3,
		"quantity": 2,
		"id": "7778a889",
		"total_income": 6,
		"updatedAt": "2023-08-11T11:46:01.159Z",
		"createdAt": "2023-08-11T11:46:01.159Z"
	}
  ```

  `400 Bad Request`
  ```json
  "Validation error: ${path} is a required field"
  ```

  ```json
  "Product already exists"
  ```

- # Update
  `PATCH /products/:id`
  ## Request example
  ```json
  {
	"name": "aa",
	"price": 6,
	"quantity": 2
  }
  ```
  
  ## Response example
  `200 OK`
  ```json
  {
	"message": "Successfully updated product",
	"update": {
		"id": "66955f2a",
		"name": "a",
		"price": 6,
		"sector": "132",
		"quantity": 1,
		"total_income": 6,
		"createdAt": "2023-08-10T14:57:14.000Z",
		"updatedAt": "2023-08-11T13:19:33.310Z"
	}
  }
  ```

  `404 Not Found`

  ```json
  "Product not found"
  ```
  ```json
  {
	"statusCode": 404,
	"message": "Cannot find this route"
  }
  ```

  `400 Bad Request`

  ```json
  "Product Name already registered"
  ```
  ```json
  "Validation error: price must be a `number` type, but the final value was: `NaN` (cast from the value `\"a\"`)."
  ```
